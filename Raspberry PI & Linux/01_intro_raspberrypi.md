# üçì Introduction to Raspberry Pi

- Raspberry Pi is a **small, affordable computer** developed for learning and projects.
- It acts as the **CPU** of your assembled computer setup.
- You connect peripherals (monitor, keyboard, mouse) to interact with it.
- It uses a **microSD card** as storage (like a hard drive).
- It has:
  - USB ports (for keyboard, mouse, etc.)
  - HDMI port (for monitor)
  - Ethernet/Wi-Fi (for internet)
  - GPIO pins (for electronics projects)
- When you set it up, you are essentially **building a computer yourself**.
